display : flex
부모요소 직계자식들을 어떤방식으로 배치할건지 정하는 속성
직계자식에게만 영향을 끼친다.(자손에게는 영향을 못준다.)

부모요소(컨테이너)의 속성
flex-direction : 자식요소 배치의 주축을 설정
justify-content : 주축을 기준으로 요소들을 어떻게 정렬할건지
align-content : 교차축을 기준으로 요소를 어떻게 정렬할건지 결정(2줄이상일 때 효과)
align-items : 교차축을 기준으로 요소를 어떻게 정렬할건지 결정(1줄이상 때 효과)
flex-wrap : 부모요소의 너비가 줄어들 때 자식 요소를 다음줄에 배치할지 설정

자식요소에 들어가는 속성
order : 정렬 순서를 변경
flex-grow : 남은 공간을 어떤비율로 차지할거냐
flex-shrink : 어떤 비율로 감소할거냐
flex-basis : 자식요소들의 기본 너비 설정
align-self : 교차 축에서 자식요소의 정렬을 개별적으로 설정

HTML과 Flexbox를 이용하면 많은 레이아웃을 만들어 볼 수 있다.
