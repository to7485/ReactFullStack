js(JavaScript)
html과 css로 만들어진 페이지에 동적인 움직임을 추가해주는 역할

화면을 조작하거나
이벤트(페이지상에서 우리가 브라우저와 하는 모든 상호작용)
처리

데이터 처리 : 계산, 조건에 따라서 코드를 실행, 반복

서버와 통신 : API 요청으로 데이터를 주고받기

우리가 만든 기능들 (로그인, 회원가입, 글쓰기, 수정,삭제,조회)

다른사람들이 만든기능

우리가 만든 화면과 기능을 연결을 해줘야 한다.

API : ~식으로 요청하면 ~형식으로 응답해줄게 라고 규칙을 정해둔것

손님 : 김치찌개 하나 주세요 -> 요청(Request)
종업원 : 주문을 받아서 주방에 전달 -> api가 서버에 요청을 전달
주방 : 김치찌개를 완성 -> 응답(Response)
종업원 : 음식을 가져다 줌 -> api를 통해 결과를 전달

html내에서는 <script> ~ </script>를 이용하여 자바스크립트 코드를 작성

head쪽에 작성 : 페이지의 초기설정이 들어간다.
body쪽에 작성 :요소의 처리

데이터 타입
현실세계의 데이터를 디지털로 표현하기 위해서 설정한 타입
숫자 : number
문자 : string
논리 : boolean (참, 거짓도 데이터로 가지고 있다)
undefined
null

변수
데이터를 저장하기 위한 컵

변수의 선언
키워드 변수명; -> 변수를 만들되, 값은 들어있지 않은 상태이다.

변수의 대입
변수명 = 값; -> 오른쪽에 있는 데이터를 왼쪽의 변수에 대입

변수의 초기화(초기값을 넣는다)
키워드 변수명 = 값;

키워드

var 변수를 재선언해도 오류가 나지 않는다.(덮어쓰기가 된다.)
값을 변수에 대입할 수 있다.

let 변수를 재선언 할 수 없다. 값을 변수에 대입할 수 있다.

const 변수를 재선언할 수 없다. 항상 초기화를 해야 한다.
값을 다시 대입할 수 없다.


타입변환(type conversion)
- 어떤 타입의 데이터를 다른 타입으로 바꾸는것

명시적 형변환
- 함수를 통해서 값을 강제로 다른 타입으로 바꾸는것
암시적 형변환
- 연산 과정에서 자연스럽게 다른 타입으로 바뀌는것

연산자
- 연산 기능이 있는 기호
- 연산을 당하는 쪽 : 피연산자

+, -, *, / , %(나머지를 구하는 연산자)


let x = 10;
let y = 7;

let result = x + y;

console.log(result);
