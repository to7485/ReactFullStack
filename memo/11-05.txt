반응형 웹

사용자가 PC, 태블릿, 모바일에서 봐도 디자인이 깨지지 않도록 해주는 것

단위 px보다는 %를 쓰거나, vw, vh, em(부모의 폰트에 비례), rem(루트의 폰트크기에 비례)

자식요소들에게 적용시키는 속성
flex : flex-grow, flex-shrink, flex-basis
   늘어나는 비율, 줄어드는 비율  기본 크기

부모요소가 가져야 하는 속성
flex-wrap : wrap; -> 너비가 좁아지면 요소들이 아래로 내려간다.

pc화면일 때 요소들이 가로로 많이 배치되도 상관이 없음

flex : 1 1 33%

태블릿 화면일 때 
flex : 1 1 45%

모바일 화면일 때(내용이 한 줄에 한개씩)
flex : 1 1 100%


@media 쿼리
화면에 너비, 높이, 해상도, 방향 등에 따라 CSS를 다르게 적용하는 문법

@media(조건){
	스타일을 정의
}

요즘은 모바일에서 웹에 접근하는일이 더 많기 때문에 모바일 우선으로 스타일을 개발하는 일이 많다.

기본스타일을 모바일에 맞게 개발하고 화면이 늘어났을 때를 미디어 쿼리를 이용하여 스타일을 변경,추가해준다.

모바일 : ~767px
태블릿 : 768 ~ 1199px
데스크탑 : 1200px ~

transition-property
어떤 CSS 속성들의 값이 바뀔 때 트래지션을 적용할 것이냐

all
현재 요소에서 값이 변하는 모든 '애니메이션 가능한' 속성에 트랜지션을 건다.
none
어떤 속성에도 트랜지션을 적용하지 않는다.
transform계열(translate, scale, rotate등)
opacity 투명도
color 글씨 색
background-color 배경색
border-color 테두리 색
box-shadow
text-shadow
width
height
등등
