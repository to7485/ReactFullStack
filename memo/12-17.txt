일반 자바스크립트 변수에 값이 바뀌어도 React는 변했다고 감지하지 못한다.

일반변수를 사용하는 경우
렌더링 결과에 영향이 없는 계산값
한 번 계산해서 JSX에 바로 쓰는 파생 값

const price = 10000;
const vat = price * 0.1;

값이 변하면 화면에도 반영이 됐으면 좋겠다~
그래서 상태(state)를 사용한다.

불변성(Immuability)
기존의 값을 직접 변경하지 않고, 새로운 값을 만들어서 교체하는 방식
기존 배역,객체를 직접 수정 -> X
기존의 값을 복사한 뒤, 변경된 새 배열, 객체를 생성 -> O

우리가 만든 프로그램을 실행하면 메모리를 점유
각 타입에 용량(메모리)

Stack(스택)
함수 호출, 지역 변수가 저장되는 공간

Heap(힙)
객체/배열이 저장되는 공간

객체가 하나 생성되면 힙공간에 자리를 차지하게 되고 객체까지 접근할 수 있는 주소를 하나 돌려준다.

주소를 변수가 알고있다.

실제 값이 아니라 값까지 접근하기 위한 주소를 알고 있는 변수
참조변수(주소를 참조하고 있다)
