git -> 개발자 사이에서 필수 도구

git init : 현재 위치를 git한테 버전관리를 부탁(최초 1번)

git add . : 내 작업 현황을 임시공간(stage)에 올려줘

git commit -m "메시지" : 임시공간에 올라와 있는 내용을 버전으로 기록해줘

git log : 지금까지 커밋(버전기록)한 현황을 보여줘

git reset --soft 해시id 8자리 : 특정 버전으로 돌아게 된다

github와 연결
git remote origin 주소

로컬 저장소에 있는 내용을 원격 저장소로 업로드
git push -u origin 브랜치명

로컬 저장소에서 개발 -> git add . -> 커밋(여기까지는 로컬 저장소) -> git push origin(원격저장소로 업로드)

개발 공부 하실 때 폴더 이름이나 파일이름은 영어로 지어주는것이 좋다.
